解决方法很简单:

1. 打开 Internet 信息服务(IIS) 管理器

2. 右击 网站 ，点击 属性

3. 切换到 HTTP 头 选项卡

4. 点击 MIME 类型 按钮

5. 点击 新建，

扩展名为 .ini

MIME类型为 application/octet-stream

6. 一路确定…搞定

IIS 下面文件打不开大多数情况下无怪乎以下两种情况：

  1. MINE 类型设置不正确
  2. 文件权限设置不正确
