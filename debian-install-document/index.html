Debian 不完全安装教程

發表於 2013-07-03   |  

准备安装介质

  1. 将 国产杂牌 USB Flash Disk 一只 插入服务器

  2. 下载 boot.img.gz，然后写入这只闪存盘

千万记得要把 sdb 换成你闪存盘的盘符，不然到时候写错了把硬盘里的东西覆盖了不要抱住我哭，我会很大笑三声然后把你踹走的233

  1. 下载 Debian Wheezy 安装镜像一只然后复制到闪存盘里

诶，等一下。

因为我们的 Debian 呢是个好孩子比较遵纪守法，不像 Ubuntu 一切为了淫民可以放弃原则什么的。

所以呢！你可能得要自己去下载一下网卡驱动… 比如我这台 R410 用的网卡驱动就没有内置，要下载一下 firmware-bnx2 这个包然后扔到闪存盘里嗯。

  1. 拔出准备安装~

安装
